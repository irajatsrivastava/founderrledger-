# 📱 FounderLedger React Native (Expo) Mobile Redesign

This directory contains the fully redesigned React Native mobile codebase for **FounderLedger**, designed according to Material Design 3 and Apple Human Interface Guidelines.

## 🚀 How to Run on Your Mobile Device

You can run this app on your physical iPhone or Android device in less than 5 minutes using **Expo Go**.

### Step 1: Pre-requisites
Make sure you have [Node.js](https://nodejs.org) installed on your development machine.

### Step 2: Install Expo CLI and Dependencies
Navigate to this directory in your terminal and install the packages:

```bash
cd src/mobile
npm install
```

### Step 3: Download the Expo Go App
On your mobile device, open the App Store (iOS) or Google Play Store (Android) and search for **"Expo Go"**. Install the free app.

### Step 4: Boot up the Developer Server
Start the local bundler:

```bash
npm start
```

This will spin up the Metro Bundler and display a QR Code in your terminal.

### Step 5: Scan the QR Code
- **Android:** Open the **Expo Go** app and tap "Scan QR Code", then point your camera at the terminal.
- **iOS:** Open your standard iOS **Camera app**, scan the QR code, and tap the notification link to open it in **Expo Go**.

Your premium redesign will load instantly with native fluid gestures and high-fidelity rendering!
